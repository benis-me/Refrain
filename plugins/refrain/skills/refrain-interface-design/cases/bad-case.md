# Bad Case

A bad Refrain interface looks like generated SaaS.

## Signals

- first viewport is a slogan or decorative hero
- feature cards replace the real workflow
- blue-purple gradient is the main style idea
- controls are text pills without hierarchy
- theme selection is a plain dropdown
- metrics are invented
- shadows do most of the separation
- only the filled state was designed
- error copy hides what failed

## Fix

Return to the product loop. Replace decorative composition with the smallest
operational surface, define semantic tokens, expose states, and add one
product-specific detail that affects use.
