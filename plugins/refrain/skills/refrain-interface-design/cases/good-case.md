# Good Case

A good Refrain interface feels like a serious tool that has already been used.

## Signals

- first viewport is the actual work surface
- controls expose the main loop without explanation
- neutral canvas, one accent, flat bordered surfaces
- design choices are visible through specimens and state
- selector rows show marks, swatches, versions, or previews
- empty and error states preserve momentum
- one small product-specific affordance proves understanding

## Example Shape

A local design generator opens to a compact composer with reference attachment,
mode, design-system selector, agent/model selector, and recent projects. The
workspace uses panes for conversation, preview, files, quality, and versions.
The design-system gallery shows component specimens, not just color chips.
