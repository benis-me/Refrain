# Selectors

Selectors are product surfaces when the choice affects output.

## Trigger

Use this file for design-system, agent, model, template, reference, asset, or
mode selectors.

## Structure

- compact trigger with icon or mark, selected value, and chevron
- two-line trigger when useful: meta label over selected value
- searchable popover when the list can grow
- built-in/custom tabs when sources differ
- selected row visible when opened
- clear, create, or rescan actions near search
- hover preview when the choice is visual

## Visual Rows

Rows should show a useful specimen, not just text:

- mark or logo
- category or provider label
- swatches or chip
- version/status when relevant
- concise description only when it changes the decision

## Reject

- plain text dropdown for visual systems
- selected item hidden below scroll
- destructive reset mixed into ordinary options
- long explanations inside every row
