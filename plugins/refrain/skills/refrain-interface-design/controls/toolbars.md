# Toolbars

Toolbars hold repeated commands close to the work surface.

## Rules

- Use icon buttons for familiar actions.
- Use icon plus text for primary or ambiguous commands.
- Use segmented controls for modes.
- Use menus for option sets.
- Keep destructive actions separated.
- Add tooltips for icon-only controls.
- Keep dimensions stable across loading and selected states.

## Common Commands

- Build
- Preview
- Scan
- Attach
- Restore
- Export
- Open log
- Rescan

## Reject

- text-pill buttons for everything
- toolbar controls that wrap unpredictably
- hidden primary actions
- controls without accessible names
